#  California Housing Price Prediction 🏠

## Project Overview

This project uses Machine Learning to predict housing prices in California based on various features such as median income, house age, number of rooms, population, and location-related attributes. The goal of this project is to build a regression model that can estimate house prices using historical housing data.

## Problem Statement

Housing price prediction is an important problem in the real estate industry. By using machine learning algorithms, we can analyze housing data and build models that estimate property prices more accurately.

The objective of this project is to train a machine learning model that predicts the median house value based on multiple housing features.

## Dataset

The dataset used in this project is the **California Housing Dataset** available in the Scikit-learn library.

Features in the dataset include:

* MedInc (Median Income)
* HouseAge
* AveRooms
* AveBedrms
* Population
* AveOccup
* Latitude
* Longitude

Target variable:

* Median House Value

## Models Used

Two regression models were used in this project:

* XGBoost Regressor
* Linear Regression

The performance of both models was evaluated using regression metrics.

## Model Evaluation Metrics

The models were evaluated using the following metrics:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)

These metrics help measure how accurately the model predicts housing prices.
